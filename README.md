Ansible role: VS Code
=========

An ansible role that installs VS Code on Linux.

Requirements
----------------

None.

Role Variables
----------------

None.

Supported distributions
----------------

This role has been been developed and tested on the following distributions :

- Fedora 43
- Fedora 42

Example Playbook
----------------

```yaml
- hosts: all
  roles:
    - xiple.vscode
```
